# Conditions

Condition (အခြေအနေ) တွေအပေါ် မူတည်ပြီး code တွေကို ဘယ်လို အလုပ်လုပ်စေမလဲဆိုတာ ဆုံးဖြတ်တဲ့နေရာမှာ သုံးပါတယ်။ ဒါကို **Control Flow** လို့လည်း ခေါ်ပါတယ်။

JavaScript မှာ condition စစ်ဆေးဖို့ နည်းလမ်းအမျိုးမျိုး ရှိပါတယ်။

- **if, else if, else**
- **Switch statement**
- **Ternary Operator**
- **Logical Short-circuiting**

---

## 1. Boolean Context

Condition တွေအားလုံးသည် နောက်ဆုံးမှာ `true` သို့မဟုတ် `false` ဆိုတဲ့ Boolean value တစ်ခုခုကို အခြေခံပြီး အလုပ်လုပ်တာဖြစ်ပါတယ်။

```js
if (true) {
  // ဒီထဲက code တွေ အလုပ်လုပ်ပါမယ်
}
```

---

## 2. Truthy and Falsy Values

JavaScript မှာ boolean မဟုတ်ပေမယ့် true သို့မဟုတ် false လို့ ယူဆရတဲ့ value တွေ ရှိပါတယ်။

- **Falsy values:** `false`, `0`, `""` (empty string), `null`, `undefined`, `NaN`
- **Truthy values:** အပေါ်က မဟုတ်တာ အားလုံး (ဥပမာ - `"hello"`, `42`, `[]`, `{}`)

---

👉 Condition တွေကို မှန်ကန်စွာ သုံးတတ်ခြင်းက programming logic တည်ဆောက်တဲ့နေရာမှာ အခြေခံအကျဆုံးဖြစ်ပါတယ်။
