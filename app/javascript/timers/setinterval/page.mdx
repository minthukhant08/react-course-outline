# setInterval

`setInterval` ကို သတ်မှတ်ထားတဲ့ အချိန်အပိုင်းအခြားတစ်ခုတိုင်းမှာ code တစ်ခုကို ထပ်ခါတလဲလဲ run စေချင်တဲ့အခါ သုံးပါတယ်။

---

## 1. Repeating Code

ဥပမာအားဖြင့် (၁) စက္ကန့်တိုင်းမှာ code တစ်ခုကို အချိန်မှန် run စေချင်တဲ့အခါ သုံးနိုင်ပါတယ်။

```js
setInterval(() => {
  console.log("၁ စက္ကန့် ပြည့်တိုင်း အလုပ်လုပ်နေပါတယ်");
}, 1000);
```

### Output

```txt
၁ စက္ကန့် ပြည့်တိုင်း အလုပ်လုပ်နေပါတယ်
၁ စက္ကန့် ပြည့်တိုင်း အလုပ်လုပ်နေပါတယ်
...
```

## 2. Stopping an interval (clearInterval)

`setInterval` ကို ပြန်ရပ်ချင်တဲ့အခါ `clearInterval()` ကို သုံးရပါတယ်။ အဲ့ဒီလို ရပ်ဖို့အတွက် `setInterval` က ပြန်ပေးတဲ့ ID ကို variable တစ်ခုထဲ သိမ်းထားဖို့ လိုပါတယ်။

```js
let count = 0;

// setInterval ရဲ့ ID ကို variable ထဲ သိမ်းမယ်
const intervalId = setInterval(() => {
  count++;
  console.log("လက်ရှိအရေအတွက်:", count);

  // ၅ ကြိမ်ပြည့်ရင် ရပ်မယ်
  if (count === 5) {
    clearInterval(intervalId);
    console.log("Interval ကို ရပ်လိုက်ပါပြီ။");
  }
}, 500);
```

### Output

```txt
လက်ရှိအရေအတွက်: 1
လက်ရှိအရေအတွက်: 2
လက်ရှိအရေအတွက်: 3
လက်ရှိအရေအတွက်: 4
လက်ရှိအရေအတွက်: 5
Interval ကို ရပ်လိုက်ပါပြီ။
```

👉 `setInterval` ကို သုံးတဲ့အခါ memory leak မဖြစ်အောင် မလိုအပ်တော့ရင် `clearInterval` နဲ့ ပြန်ပိတ်ဖို့ မမေ့ပါနဲ့။
