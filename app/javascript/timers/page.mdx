# Timers

JavaScript မှာ code တွေကို အခုချက်ချင်း run တာမျိုးမဟုတ်ဘဲ အချိန်တစ်ခု စောင့်ပြီးမှ run ချင်တာ ဒါမှမဟုတ် ပုံမှန်အချိန်ခြားပြီး ထပ်ခါတလဲလဲ run ချင်တာမျိုးတွေအတွက် Timer တွေကို သုံးပါတယ်။
---

## 1. Why Timers?
Timer တွေက JavaScript ရဲ့ asynchronous feature တစ်ခုဖြစ်ပါတယ်။ သူတို့က program ကို ရပ်မထားဘဲ (non-blocking) နောက်ကွယ်ကနေ အချိန်မှတ်ပေးထားပြီး သတ်မှတ်ချိန်ရောက်မှ သက်ဆိုင်ရာ code တွေကို run ပေးတာ ဖြစ်ပါတယ်။

အဓိက သုံးလေ့ရှိတဲ့ Timer (၂) မျိုး ရှိပါတယ် -
- `setTimeout`: အချိန်တစ်ခု စောင့်ပြီးမှ code ကို (၁) ကြိမ်ပဲ run စေချင်တဲ့အခါ သုံးတယ်။
- `setInterval`: အချိန်တစ်ခုခြားပြီး code ကို ထပ်ခါတလဲလဲ run စေချင်တဲ့အခါ သုံးတယ်။
