# Comparison Operators

Value နှစ်ခုကို နှိုင်းယှဉ်ပြီး မှန် (true) သို့မဟုတ် မှား (false) ဆိုတဲ့ Boolean ရလဒ် ထုတ်ပေးပါတယ်။

---

## Equality Operators

| Operator | Description | Example |
|----------|-------------|---------|
| `==` | Equal to (value ပဲတူရင်ရပြီ) | `5 == "5" // true` |
| `===` | Strict equal (value ရော type ရော တူရမယ်) | `5 === "5" // false` |
| `!=` | Not equal | `5 != 8 // true` |
| `!==` | Strict not equal | `5 !== "5" // true` |

---

## Relational Operators

| Operator | Description | Example |
|----------|-------------|---------|
| `>` | Greater than | `10 > 5 // true` |
| `<` | Less than | `10 < 5 // false` |
| `>=` | Greater than or equal to | `10 >= 10 // true` |
| `<=` | Less than or equal to | `5 <= 10 // true` |

---

👉 JavaScript မှာ coding ရေးတဲ့အခါ ပိုပြီး တိကျသေချာစေဖို့ `===` (Strict equality) ကိုပဲ အသုံးများပါတယ်။
