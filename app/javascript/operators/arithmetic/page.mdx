# Arithmetic Operators

ကိန်းဂဏန်းတွေကို တွက်ချက်တဲ့နေရာမှာ သုံးပါတယ်။

---

## Basic Operators

| Operator | Description | Example |
|----------|-------------|---------|
| `+` | Addition (ပေါင်းခြင်း) | `10 + 5 = 15` |
| `-` | Subtraction (နှုတ်ခြင်း) | `10 - 5 = 5` |
| `*` | Multiplication (မြှောက်ခြင်း) | `10 * 5 = 50` |
| `/` | Division (စားခြင်း) | `10 / 5 = 2` |
| `%` | Remainder (အကြွင်းရှာခြင်း) | `10 % 3 = 1` |
| `**` | Exponentiation (ထပ်ညွှန်း) | `2 ** 3 = 8` |

---

## Increment & Decrement

| Operator | Description | Example |
|----------|-------------|---------|
| `++` | Increment (၁ တိုးခြင်း) | `x++` |
| `--` | Decrement (၁ လျှော့ခြင်း) | `x--` |

```js
let x = 5;
x++; 
console.log(x); // 6

let y = 10;
y--;
console.log(y); // 9
```

---

👉 `+` operator ကို string တွေ ပေါင်းတဲ့နေရာ (Concatenation) မှာလည်း သုံးနိုင်ပါတယ်။
