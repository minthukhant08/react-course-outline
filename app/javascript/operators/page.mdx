# Operators

Operator ဆိုတာ variable တွေနဲ့ value တွေကို calculation ဒါမှမဟုတ် comparison (နှိုင်းယှဉ်ခြင်း) လုပ်တဲ့အခါ သုံးတဲ့ သင်္ကေတ (symbols) တွေဖြစ်ပါတယ်။

JavaScript မှာ အသုံးများတဲ့ operator အမျိုးအစားတွေကတော့ -

- **Arithmetic Operators** `(+, -, *, /, ...)`
- **Assignment Operators** `(=, +=, -=, ...)`
- **Comparison Operators** `(==, ===, >, <, ...)`
- **Logical Operators** `(&&, ||, !)`
- **Type Operators** `(typeof)`

---

## 1. Operands and Operators

Operator တွေနဲ့ အလုပ်လုပ်တဲ့ value တွေကို **Operand** လို့ ခေါ်ပါတယ်။

```js
let x = 10 + 5;
```

ဒီနေရာမှာ `+` သည် **Operator** ဖြစ်ပြီး `10` နဲ့ `5` သည် **Operand** တွေဖြစ်ပါတယ်။

---

## 2. Operator Precedence

တွက်ချက်မှုတစ်ခုတည်းမှာ operator တွေ အများကြီးပါလာရင် ဘယ်ဟာကို အရင်တွက်ရမလဲဆိုတာ သတ်မှတ်ထားတဲ့ precedence ရှိပါတယ်။ (ဥပမာ - အမြှောက်/အစားကို အရင်တွက်ခြင်း)

```js
let res = 10 + 5 * 2; // 20
```

---

👉 Operator တွေကို နားလည်ထားခြင်းက JavaScript logic တွေကို တည်ဆောက်တဲ့နေရာမှာ အခြေခံအကျဆုံးနဲ့ အရေးကြီးဆုံး အစိတ်အပိုင်းတစ်ခုဖြစ်ပါတယ်။
