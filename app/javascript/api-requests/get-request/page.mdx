# GET Request

GET request ကို server ဆီကနေ data တွေကို ဖတ်ယူချင်တဲ့အခါ သုံးပါတယ်။ ဒါဟာ `fetch()` ရဲ့ default method လည်း ဖြစ်ပါတယ်။

---

## 1. Using async/await with Fetch

`fetch()` ကို `.then()` နဲ့ ရေးလို့ရသလို `async/await` နဲ့ ရေးတာက ပိုပြီး ဖတ်ရလွယ်ကူပါတယ်။

```js
async function getUser() {
  const response = await fetch('https://jsonplaceholder.typicode.com/users/1');
  const data = await response.json();
  
  console.log("name:", data.name);
  console.log("Email:", data.email);
}

getUser();
```

### Output

```txt
name: Leanne Graham
Email: Sincere@april.biz
```

👉 GET request ကိုသုံးတဲ့အခါ body ထည့်ပေးစရာ မလိုပါဘူး။
