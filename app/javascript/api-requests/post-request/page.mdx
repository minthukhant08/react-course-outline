# POST Request

POST request ကို server ဆီကို data အသစ်တွေ ပေးပို့ချင်တဲ့အခါ (ဥပမာ- user register လုပ်တာ၊ post အသစ်တင်တာ) သုံးပါတယ်။

---

## 1. Sending Data

POST လုပ်တဲ့အခါ `method`, `headers` နဲ့ `body` ဆိုတဲ့ အချက်အလက်တွေကို ထည့်ပေးဖို့ လိုပါတယ်။

```js
async function createPost() {
  const response = await fetch('https://jsonplaceholder.typicode.com/posts', {
    method: 'POST',
    body: JSON.stringify({
      title: 'စာမူအသစ်',
      body: 'ဒါကတော့ ကျောင်းသားများအတွက် test ပါ။',
      userId: 1,
    }),
    headers: {
      'Content-type': 'application/json; charset=UTF-8',
    },
  });

  const data = await response.json();
  console.log("ပို့လိုက်တဲ့ data:", data);
}

createPost();
```

### Output

```txt
ပို့လိုက်တဲ့ data: { title: 'စာမူအသစ်', body: '...', userId: 1, id: 101 }
```

👉 `JSON.stringify()` ကို သုံးပြီး object ကို string အဖြစ် ပြောင်းလဲပေးပို့ရပါတယ်။
