# HTTP Verbs

HTTP Verbs (သို့မဟုတ်) HTTP Methods ဆိုတာ client (browser) နဲ့ server တို့ကြား ဆက်သွယ်တဲ့အခါ ဘယ်လိုအလုပ်မျိုး လုပ်ချင်တာလဲဆိုတာကို သတ်မှတ်ပေးတဲ့ standard နည်းလမ်းတွေ ဖြစ်ပါတယ်။

---

## 1. What are HTTP Verbs?

Website တစ်ခုကို ကြည့်တဲ့အခါ၊ login ဝင်တဲ့အခါ ဒါမှမဟုတ် post တစ်ခု တင်တဲ့အခါ နောက်ကွယ်မှာ HTTP request တွေကို သုံးရပါတယ်။ အဲ့ဒီ request တွေက ဘာကို ဆိုလိုတာလဲဆိုတာ သိစေဖို့ HTTP Verbs တွေကို သုံးပါတယ်။

အသုံးအများဆုံး Verbs တွေကတော့ -
- **GET**: Data ရယူရန်
- **POST**: Data အသစ်ပို့ရန်
- **PUT / PATCH**: ရှိပြီးသား data ကို ပြင်ရန်
- **DELETE**: Data ဖျက်ရန်

👉 HTTP Verbs တွေကို မှန်ကန်စွာ သုံးခြင်းက API တစ်ခုကို ပိုမိုစနစ်တကျ ရှိစေပါတယ်။
