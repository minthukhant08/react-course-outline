# DELETE Method

DELETE method ကို server ပေါ်က resource (data) တစ်ခုခုကို ဖျက်ဆီး (delete) ချင်တဲ့အခါမှာ သုံးပါတယ်။

---

## 1. Using DELETE

DELETE request ပို့တဲ့အခါ ဘယ်အရာကို ဖျက်ချင်တာလဲဆိုတဲ့ ID သို့မဟုတ် Unique key တစ်ခုခုကို URL မှာ ထည့်ပေးရလေ့ ရှိပါတယ်။

```text
DELETE /api/users/1
```

အကယ်၍ အောင်မြင်စွာ ဖျက်နိုင်ခဲ့ရင် server က များသောအားဖြင့် `200 OK` ဒါမှမဟုတ် `204 No Content` ဆိုတဲ့ status code ပြန်ပေးတတ်ပါတယ်။

👉 DELETE method သည် idempotent ဖြစ်ပါသည်။ ဆိုလိုသည်မှာ တစ်ခုတည်းကို အကြိမ်ကြိမ် ဖျက်ခိုင်းသော်လည်း ရလဒ်မှာ ပြောင်းလဲမှု မရှိပါ။
