# JavaScript

JavaScript ဆိုတာ modern web development ရဲ့ အခြေခံအကျဆုံးနဲ့ အရေးကြီးဆုံး language တစ်ခုဖြစ်ပါတယ်။ 

ဒီ section မှာ JavaScript ရဲ့ အခြေခံ syntax များ၊ logic များနဲ့ advanced concepts တွေကို အသေးစိတ် လေ့လာနိုင်ပါတယ်။

---

## What you will learn:

- **Variables & Data Types**: Data တွေကို ဘယ်လိုသိမ်းမလဲ။
- **Loops & Conditions**: Logic တွေကို ဘယ်လိုကိုင်တွယ်မလဲ။
- **Arrays & Objects**: Data collections တွေကို စနစ်တကျသုံးခြင်း။
- **Promises & Async**: Asynchronous အလုပ်တွေကို ကိုင်တွယ်ခြင်း။
