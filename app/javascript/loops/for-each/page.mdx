# Array.forEach()

`.forEach()` method သည် array ထဲက element တစ်ခုချင်းစီအတွက် function တစ်ခုကို အလုပ်လုပ်စေပါတယ်။

---

## Basic Usage

```js
const colors = ["Red", "Green", "Blue"];

colors.forEach((color, index) => {
  console.log(index + ": " + color);
});
```

### Output
```txt
0: Red
1: Green
2: Blue
```

---

## Difference from for/for-of

- `.forEach()` သည် functional style ဖြစ်ပါတယ်။
- `.forEach()` ထဲမှာ `break` သို့မဟုတ် `continue` သုံးလို့ မရပါဘူး။
- ဘာရလဒ်မှ return ပြန်မပေးပါဘူး (`undefined` သာ ပြန်ပေးပါတယ်)။

---

👉 Array ထဲက element တွေကို တစ်ခုခု (ဥပမာ - log ထုတ်ခြင်း၊ API ခေါ်ခြင်း) လုပ်ဆောင်ချင်ရုံ သက်သက်ဆိုရင် `.forEach()` ကို သုံးပါတယ်။
