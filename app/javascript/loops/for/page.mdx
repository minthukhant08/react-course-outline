# For Loop

`for` loop သည် အကြိမ်အရေအတွက် အတိအကျ သိတဲ့အခါမှာ သုံးဖို့ အသင့်တော်ဆုံး ဖြစ်ပါတယ်။

---

## Syntax

```js
for (initialization; condition; increment/decrement) {
  // code block to be executed
}
```

---

## Example

```js
for (let i = 0; i < 5; i++) {
  console.log("Count is: " + i);
}
```

### Output
```txt
Count is: 0
Count is: 1
Count is: 2
Count is: 3
Count is: 4
```

---

## How it works?
1. **initialization**: loop မစခင် တစ်ကြိမ်ပဲ အလုပ်လုပ်ပါတယ်။ (ဥပမာ - `let i = 0`)
2. **condition**: loop တစ်ပတ်တိုင်းမှာ စစ်ဆေးပါတယ်။ `true` ဖြစ်နေသရွေ့ loop ဆက်ပတ်ပါမယ်။
3. **increment**: loop တစ်ပတ်ပြီးတိုင်း variable ကို တိုးပေးပါတယ်။

---

👉 Array ထဲက element တွေကို index နဲ့ ပတ်ချင်တဲ့အခါ `for` loop ကို အသုံးများပါတယ်။
