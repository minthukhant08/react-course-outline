# Break & Continue

Loop တွေရဲ့ ပုံမှန် အလုပ်လုပ်ပုံကို ထိန်းချုပ်ဖို့ `break` နဲ့ `continue` ကို သုံးပါတယ်။

---

## 1. Break

`break` သည် loop ကို ချက်ချင်း ရပ်တန့်ပြီး အပြင်ကို ထွက်လိုက်ပါတယ်။

```js
for (let i = 1; i <= 5; i++) {
  if (i === 3) {
    break; // 3 ရောက်ရင် loop ရပ်မယ်
  }
  console.log(i);
}
```

### Output
```txt
1
2
```

---

## 2. Continue

`continue` သည် လက်ရှိ အကြိမ် (current iteration) ကိုပဲ ကျော်သွားပြီး နောက်တစ်ပတ်ကို ဆက်သွားပါတယ်။

```js
for (let i = 1; i <= 5; i++) {
  if (i === 3) {
    continue; // 3 ကို ကျော်သွားမယ်
  }
  console.log(i);
}
```

### Output
```txt
1
2
4
5
```

---

👉 `break` ကို ရှာနေတဲ့ data တွေ့သွားလို့ loop ဆက်မပတ်ချင်တဲ့အခါ သုံးပြီး၊ `continue` ကိုတော့ မလိုအပ်တဲ့ data တွေကို ကျော်ချင်တဲ့အခါ သုံးပါတယ်။
