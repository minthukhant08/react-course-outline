# While Loop

`while` loop သည် condition တစ်ခု `true` ဖြစ်နေသရွေ့ code block ကို အကြိမ်ကြိမ် အလုပ်လုပ်စေပါတယ်။

---

## Syntax

```js
while (condition) {
  // code block
}
```

---

## Example

```js
let i = 0;

while (i < 3) {
  console.log("i is " + i);
  i++; // သတိပြုရန် - i ကို မတိုးပေးရင် infinite loop ဖြစ်သွားပါမယ်
}
```

### Output
```txt
i is 0
i is 1
i is 2
```

---

👉 `while` loop ကို အကြိမ်အရေအတွက် အတိအကျ မသိဘဲ condition တစ်ခုအပေါ်ပဲ မူတည်ပြီး loop ပတ်ချင်တဲ့အခါ သုံးပါတယ်။
