# For In Loop

`for...in` loop ကို **Object** ထဲမှာရှိတဲ့ property (keys) တွေကို တစ်ခုချင်းစီ ထုတ်ကြည့်ချင်တဲ့အခါ သုံးပါတယ်။

---

## Example with Object

```js
const user = {
  name: "Mg Mg",
  age: 20,
  city: "Yangon"
};

for (let key in user) {
  console.log(key + ": " + user[key]);
}
```

### Output
```txt
name: Mg Mg
age: 20
city: Yangon
```

---

## Important Note

`for...in` ကို array တွေမှာလည်း သုံးလို့ရပေမယ့် index တွေကိုပဲ ထုတ်ပေးတာဖြစ်ပြီး order မမှန်နိုင်တဲ့အတွက် array တွေအတွက်တော့ `for...of` ကိုပဲ သုံးသင့်ပါတယ်။

---

👉 Object ရဲ့ keys တွေကို iterate လုပ်ဖို့အတွက် `for...in` က အကောင်းဆုံး ဖြစ်ပါတယ်။
