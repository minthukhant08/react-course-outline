
# Accessing Object Values

Object ထဲက data တွေကို access လုပ်ဖို့ နည်းလမ်း ၂ မျိုးရှိပါတယ်။

---

## 1. Dot Notation

```js
const user = {
  name: "Mg Mg",
  age: 25
};

console.log(user.name);
console.log(user.age);
```

#### Output

```txt
Mg Mg
25
```

👉 အသုံးအများဆုံးနည်းလမ်းဖြစ်ပါတယ်။

---

## 2. Bracket Notation

```js
const user = {
  name: "Mg Mg",
  age: 25
};

console.log(user["name"]);
console.log(user["age"]);
```

### Output

```txt
Mg Mg
25
```

👉 Dynamic key တွေအတွက် bracket notation ကိုအသုံးများပါတယ်။
