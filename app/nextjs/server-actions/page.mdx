# Overview

Server Actions ဆိုတာ server ပေါ်မှာ တိုက်ရိုက် run တဲ့ asynchronous functions တွေ ဖြစ်ပါတယ်။ API routes တွေ သီးခြားရေးစရာမလိုဘဲ data mutations (ဥပမာ - create, update, delete) တွေကို component ထဲကနေ တိုက်ရိုက် ခေါ်ယူနိုင်ပါတယ်။

## Contents
- **Defining Actions**: Server Action တည်ဆောက်ခြင်း။
- **Using in Forms**: Forms တွေနဲ့ တွဲသုံးခြင်း။
- **Client Components**: Client components ကနေ လှမ်းခေါ်ခြင်း။
- **Revalidate & Redirect**: အလုပ်လုပ်ပြီးရင် data update လုပ်နည်းနဲ့ page redirect ခြင်း။

👉 Server Actions တွေက client နဲ့ server ကြား ဆက်သွယ်ရေးကို အလွန်ရိုးရှင်းစေပါတယ်။
