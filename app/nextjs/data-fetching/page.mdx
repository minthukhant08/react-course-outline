# Overview

Next.js မှာ data တွေကို ဘယ်လို fetch လုပ်မလဲနဲ့ data တွေကို ပိုမိုမြန်ဆန်အောင် ဘယ်လို cache လုပ်မလဲဆိုတာဟာ application တစ်ခုရဲ့ စွမ်းဆောင်ရည်အတွက် အရေးကြီးပါတယ်။

## Contents
- **Server Fetching**: Server Components ထဲမှာ data ယူခြင်း
- **Caching Options**: `force-cache` နဲ့ `no-store` 
- **Revalidation**: Cache data တွေကို update လုပ်ခြင်း
- **Tags & On-demand**: Tags သုံးပြီး လိုအပ်တဲ့အချိန်မှ cache ဖျက်ခြင်း။
- **Modern Caching**: `"use cache"` directive အသစ်အကြောင်း

👉 Data Fetching strategy ကို မှန်ကန်စွာ ရွေးချယ်ခြင်းဖြင့် website ကို ပိုမိုမြန်ဆန်ချောမွေ့စေမှာ ဖြစ်ပါတယ်။
