# Overview

Next.js မှာ website တစ်ခုရဲ့ performance နဲ့ user experience ကို ပိုမိုကောင်းမွန်စေဖို့အတွက် optimization တွေ အလိုအလျောက်ပါဝင်တဲ့ built-in components တွေကို support ပေးထားပါတယ်။

## Contents
- **Link**: Client-side navigation နဲ့ prefetching လုပ်ခြင်း
- **Image**: Image optimization နဲ့ performance လုပ်ခြင်း
- **Form**: Search forms တွေမှာ URL params handling လုပ်ခြင်း
- **Font & Script**: Web fonts နဲ့ third-party scripts optimize လုပ်ခြင်း

👉 Next.js built-in components တွေကို သုံးစွဲခြင်းက website ကို [Core Web Vitals standard](https://developers.google.com/search/docs/appearance/core-web-vitals) တွေနဲ့ ကိုက်ညီအောင် ကူညီပေးပါတယ်။
