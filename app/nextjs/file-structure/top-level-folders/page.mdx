# Top-level Folders

Next.js project တစ်ခုရဲ့ root level မှာ အဓိက ပါဝင်လေ့ရှိတဲ့ folders များမှာ အောက်ပါအတိုင်း ဖြစ်ပါတယ်။

---

## 1. `app/` Directory

App Router အတွက် အဓိက အသုံးပြုတဲ့ directory ဖြစ်ပါတယ်။ Routing, Layouts နဲ့ Pages တွေ အားလုံး ဒီထဲမှာ ရှိပါတယ်။ Next.js (version 13+) ရဲ့ modern routing စနစ် ဖြစ်ပါတယ်။

---

## 2. `pages/` Directory (Legacy)

Pages Router အတွက် အသုံးပြုပါတယ်။ App Router မပေါ်ခင်က သုံးခဲ့တဲ့ နည်းလမ်းဖြစ်ပြီး ယနေ့တိုင် project အဟောင်းတွေမှာ တွေ့နိုင်ပါတယ်။

---

## 3. `public/` Folder

Static assets များဖြစ်တဲ့ images, fonts, icons တွေကို သိမ်းဆည်းဖို့ သုံးပါတယ်။ ဒီထဲက file တွေကို `/` path နဲ့ တိုက်ရိုက် လှမ်းယူနိုင်ပါတယ်။ (ဥပမာ - `public/logo.png` ကို `/logo.png` လို့ သုံးနိုင်ပါတယ်)

---

## 4. `src/` Directory (Optional)

အကယ်၍ project root မှာ config files တွေ အရမ်းများနေရင် code တွေကို သီးခြားစီ ခွဲထားချင်တဲ့အခါ application code အားလုံး (app folder အပါအဝင်) ကို `src` folder ထဲမှာ ထည့်သွင်းနိုင်ပါတယ်။ (ဥပမာ - `src/app/`)

👉 Folders တွေရဲ့ တည်နေရာကို သိရှိထားခြင်းဖြင့် project ကို စနစ်တကျ စီမံနိုင်မှာ ဖြစ်ပါတယ်။
