# Overview

Next.js မှာ project တစ်ခုကို စနစ်တကျ တည်ဆောက်ဖို့အတွက် သတ်မှတ်ထားတဲ့ folder နဲ့ file convention တွေ ရှိပါတယ်။ ဒါတွေကို နားလည်ထားခြင်းဖြင့် project organization ကို ပိုမိုကောင်းမွန်စေမှာ ဖြစ်ပါတယ်။

## Contents
- **Top-level Folders**: Project ရဲ့ အဓိက folders များ
- **Special Files**: App Router မှာသုံးတဲ့ အထူး file များ
- **Organization**: Private folders, Route groups နဲ့ Dynamic routes များ
