# TypeScript Overview

TypeScript ဆိုတာ JavaScript ပေါ်မှာ ထပ်ဆင့်တည်ဆောက်ထားတဲ့ open-source language တစ်ခုဖြစ်ပါတယ်။ သူက JavaScript ရဲ့ feature အားလုံးကို အသုံးပြုနိုင်တဲ့အပြင် "Static Typing" ဆိုတဲ့ အားသာချက်ကိုပါ ပေါင်းထည့်ပေးထားပါတယ်။

---

## 1. How TypeScript helps Developers?

TypeScript ရဲ့ အဓိက ရည်ရွယ်ချက်ကတော့ code ရေးနေစဉ်မှာတင် အမှားတွေကို ကြိုတင်သိရှိစေဖို့နဲ့ code ရဲ့ quality ကို မြှင့်တင်ပေးဖို့ ဖြစ်ပါတယ်။

### Catching Errors Early
JavaScript မှာဆိုရင် variable တစ်ခုရဲ့ data type က ကြိုက်သလို ပြောင်းလဲသွားနိုင်တဲ့အတွက် runtime ရောက်မှ error တက်တာမျိုး ဖြစ်တတ်ပါတယ်။ TypeScript မှာတော့ data type ကို တိတိကျကျ သတ်မှတ်ထားနိုင်တဲ့အတွက် မှားယွင်းတဲ့ data အမျိုးအစား ထည့်မိတာနဲ့ ချက်ချင်း သိနိုင်ပါတယ်။

```ts
// JavaScript မှာဆိုရင် ဒါက runtime မှ error တက်မယ်
function greet(name) {
  return "Hello, " + name.toUpperCase();
}

// TypeScript မှာဆိုရင် name က string ဖြစ်ရမယ်လို့ သတ်မှတ်လိုက်မယ်
function greetTS(name: string) {
  return "Hello, " + name.toUpperCase();
}

// greetTS(123); // ❌ ဒါဆိုရင် TypeScript က ချက်ချင်း error ပြပါလိမ့်မယ်
```

### Better IntelliSense & Auto-completion
Data type တွေကို သတ်မှတ်ထားတဲ့အတွက် VS Code လိုမျိုး code editor တွေမှာ function တွေ၊ object properties တွေကို ပိုမိုတိကျစွာ ညွှန်းဆိုပေးနိုင် (suggest လုပ်ပေးနိုင်) ပါတယ်။

## 2. Basic Example

TypeScript မှာ variable တစ်ခုကို data type နဲ့အတူ အခုလို ကြေညာနိုင်ပါတယ်။

```ts
let age: number = 25;
let username: string = "Mg Mg";
let isStudent: boolean = true;

function add(a: number, b: number): number {
  return a + b;
}

console.log(add(10, 20));
```

### Output

```txt
30
```
