# Introduction

## 1. Javascript   
Modern JS style တွေကို အရင် အကျွမ်းတဝင်ရှိဖို့လိုပါမယ်။ `map, filter, anonymous functions declaration, module import/export` etc... ရေးတဲ့ style တွေအမျိုးမျိုးရှိလို့ သူများ code တွေ ဖတ်တဲ့အခါ အခက်အခဲမဖြစ်နိုင်ဖို့လိုပါတယ်။

## 2. Typescript
သူကတော့ `runtime` အတွက် မလိုပေမဲ့ `development` အတွက် `type safety` အရမ်းကောင်းပါတယ်။ တော်ရုံသတိမထားမိပဲ `runtime` မှတက်နိုင်တဲ့ `error` တွေကို `code` ရေးနေချိန်မှာပဲ `error` ပြပြီး `type safety` ကို `enforce` လုပ်ပေးတယ်။
