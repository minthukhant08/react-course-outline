import { Sandpack } from "@codesandbox/sandpack-react";

# Dynamic Values in JSX

JSX ထဲမှာ JavaScript expressions တွေကို ထည့်သွင်းအသုံးပြုချင်တဲ့အခါ curly braces `{}` ကို အသုံးပြုရပါတယ်။ ဒါဟာ static HTML ကနေ dynamic ဖြစ်တဲ့ UI အဖြစ် ပြောင်းလဲပေးတဲ့ အဓိက နည်းလမ်းဖြစ်ပါတယ်။

---

## 1. Using Variables

Variables တွေထဲမှာ သိမ်းထားတဲ့ data တွေကို JSX ထဲမှာ တိုက်ရိုက် ထည့်သွင်းပြသနိုင်ပါတယ်။

<Sandpack 
  template="react-ts"
  theme="dark"
  files={{
    "/App.tsx": `export default function App() {
  const name = "Aung Aung";
  const age = 20;

  return (
    <div>
      <h1>Name: {name}</h1>
      <p>Age: {age}</p>
    </div>
  );
}`,
  }}
/>

### Output
```txt
Name: Aung Aung
Age: 20
```

---

## 2. Expressions and Calculations

Curly braces ထဲမှာ JavaScript expressions တွေကို တိုက်ရိုက် ရေးသားနိုင်ပါတယ်။

<Sandpack 
  template="react-ts"
  theme="dark"
  files={{
    "/App.tsx": `export default function App() {
  const x = 10;
  const y = 20;

  return (
    <div>
      <p>Total: {x + y}</p>
      <p>Is 10 greater than 5? {10 > 5 ? "Yes" : "No"}</p>
    </div>
  );
}`,
  }}
/>

### Output
```txt
Total: 30
Is 10 greater than 5? Yes
```

---

## 3. Function Calls

Function တစ်ခုကို လှမ်းခေါ်ပြီး သူ့ရဲ့ return value ကိုလည်း JSX ထဲမှာ ပြသနိုင်ပါတယ်။

<Sandpack 
  template="react-ts"
  theme="dark"
  files={{
    "/App.tsx": `export default function App() {
  const getGreeting = (time: string) => {
    return "Good " + time;
  };

  return (
    <div>
      <h2>{getGreeting("Morning")}</h2>
      <h2>{getGreeting("Evening")}</h2>
    </div>
  );
}`,
  }}
/>

### Output
```txt
Good Morning
Good Evening
```

👉 JSX ရဲ့ attributes (ဥပမာ `src`, `href`) တွေမှာလည်း dynamic values သုံးမယ်ဆိုရင် quotation marks `""` အစား `{}` ကို သုံးရပါတယ်။
