# ReactJS Overview

React ဆိုတာ UI (User Interface) တွေကို တည်ဆောက်ဖို့အတွက် Facebook (Meta) က ထုတ်လုပ်ထားတဲ့ အသုံးအများဆုံး JavaScript library တစ်ခု ဖြစ်ပါတယ်။

Website တစ်ခုရဲ့ UI (View layer) ကို ကိုင်တွယ်တဲ့နေရာမှာ component-based architecture ကို အသုံးပြုပါတယ်။ ဆိုလိုတာက website တစ်ခုလုံးကို အစိတ်အပိုင်းလေးတွေ (Components) ခွဲပြီး စနစ်တကျ တည်ဆောက်တာ ဖြစ်ပါတယ်။

React ရဲ့ အဓိက ထူးခြားချက်များ -
- **Declarative**: React က UI ကို ဘယ်လိုပုံစံ ဖြစ်ချင်တာလဲဆိုတာပဲ ပြောရုံပါပဲ။ အပြောင်းအလဲတွေကို React က နောက်ကွယ်ကနေ အလိုအလျောက် ကိုင်တွယ်ပေးပါတယ်။
- **Component-Based**: Code တွေကို တစ်ခါရေးထားပြီး နေရာအနှံ့မှာ ပြန်သုံးနိုင်တဲ့ (Reusable) components တွေအဖြစ် တည်ဆောက်နိုင်ပါတယ်။
---

## Basic Example (TypeScript)
TypeScript နဲ့ တွဲဖက်အသုံးပြုခြင်းအားဖြင့် ပိုမို robust ဖြစ်ပြီး error ကင်းတဲ့ application တွေကို တည်ဆောက်နိုင်ပါတယ်။

React မှာ code တွေကို TSX (TypeScript XML) ပုံစံနဲ့ ရေးသားပါတယ်။ ဒါဟာ TypeScript ထဲမှာ HTML ရေးသလိုမျိုး ဖြစ်ပါတယ်။

<Sandpack 
  template="react-ts"
  theme="dark"
  files={{
    "/App.tsx": `export default function App() {
  return <h1>Your first React component with TypeScript!</h1>;
}`,
  }}
/>

👉 React ကို TypeScript နဲ့ တွဲသုံးတာဟာ အကြီးစား project တွေအတွက် industry standard တစ်ခု ဖြစ်ပါတယ်။
